\# Data



This folder contains the source datasets used in the applied NLP text classification case study.



\## Files



| File | Description | Use in project |

|---|---|---|

| `raw/movie\_data.csv` | Movie review text with sentiment labels. | Used for sentiment classification and model evaluation. |

| `raw/emotions/train.txt` | Training split for emotion-labeled text. | Used for emotion classification modeling. |

| `raw/emotions/val.txt` | Validation split for emotion-labeled text. | Used for model validation. |

| `raw/emotions/test.txt` | Test split for emotion-labeled text. | Used for final model evaluation. |

| `raw/image\_caption.txt` | Caption-style text data with image IDs. | Used for text preprocessing and exploratory NLP analysis. |



\## Notes



The datasets are included to make the notebook reproducible and easier to review.



This project is presented as an applied NLP portfolio case study focused on text preprocessing, feature extraction, classification modeling, evaluation, and decision-support interpretation.
